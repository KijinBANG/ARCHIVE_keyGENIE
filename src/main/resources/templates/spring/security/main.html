Spring Boot Security

for All for Guest for Member for Admin
※ Security 개요
  1. 개발 주안점
    • 위와 같이 세가지 계층 설정
    • 로그인하면 기본적으로 GUEST 권한
    • 방명록은 아무 권한 없어도 CRUD 가능
    • 게시판은 GUEST도 읽을 수 있음(Register 버튼은 Memeber 이상의 권한을 가진 사용자에게만 노출)
    • 권한이 없는 경우, 관리자에게 문의하도록 유도!
  2. 사용방법
    • 오른쪽 사이드바에 뜨는 권한을 확인
    • 위의 버튼을 클릭 해 보면서 기능이 잘 구현되었는지 확인
    • 정식 배포 전에 전체 삭제해야할지? 아님 다른 방법을 고려해야할지?
  3. 해얄 할 작업
    • 접근 권한이 없어서 에러발생했을 때 뜨는 창 기획할 것!
